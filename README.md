# KR Actual Salary (한국 연봉 실수령액 계산기)

한국의 연봉을 입력받아 월/연 실수령액을 정확하게 계산하는 CLI 도구입니다.

## 📋 개요

이 도구는 한국의 세법과 사회보험 제도를 기반으로 연봉에서 실제 받을 수 있는 실수령액을 계산합니다. 2025년 기준의 세율과 사회보험료율을 적용하여 정확한 계산을 제공합니다.

## 주요 기능

- **간편한 입력**: 연봉을 만원 단위로 입력 (예: 5000 → 5,000만원)
- **정확한 계산**: 2025년 기준 세법 및 사회보험 제도 반영
- **상세한 내역**: 각종 공제 항목별 상세 내역 제공
- **CLI 인터페이스**: 터미널에서 바로 사용 가능

## 계산 항목

### 사회보험 (근로자 부담)
- **국민연금**: 4.5% (월 39만원~590만원 구간)
- **건강보험**: 3.545%
- **장기요양보험**: 건강보험료의 12.95%
- **고용보험**: 0.9%

### 세금
- **소득세**: 누진공제식 (6%~45% 구간별 세율)
- **지방소득세**: 소득세의 10%
- **근로소득세액공제**: 산출세액의 55% (한도 적용)

### 공제
- **비과세 수당**: 월 20만원 (식대)
- **근로소득공제**: 총급여 기준 구간별 공제
- **기본공제**: 1인당 150만원

## 🚀 설치 및 사용법

### 설치

```bash
# 프로젝트 클론
git clone <repository-url>
cd kr-actual-salary

# 의존성 설치
yarn install

# 빌드
yarn build
```

### 사용법

```bash
# 개발 모드로 실행
yarn dev

# 빌드된 버전 실행
yarn start
```

실행 후 연봉을 만원 단위로 입력하세요:

```
연봉을 "만원 단위"로 입력해주세요 (예: 5000) > 5000
```
